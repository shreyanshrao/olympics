# 🥇 Medal Winners Visualization Dashboard

This project analyzes and visualizes a dataset containing **medal-winning athletes**, their **countries**, and the **sports** in which they won medals.

---

## 📊 Features

This project includes visualizations like:

1. **Bar Plot** - Medals per Country  
2. **Pie Chart** - Distribution of Medals by Sport  
3. **Heatmap** - Country vs Sport Medal Concentration  
4. **Bar Plot** - Top Medal-Winning Athletes  
5. **Stacked Bar Plot** - Medal Distribution by Sport per Country  

---

## 📁 Dataset Format

The dataset should be a CSV file with the following columns:

| Name      | Country     | Sport      |
|-----------|-------------|------------|
| Alice     | USA         | Swimming   |
| Bob       | India       | Wrestling  |
| Clara     | USA         | Athletics  |

> 🎯 One row per medal-winning athlete.
